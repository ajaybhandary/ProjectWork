RhineSignup This is a BDD framework using cucumber and Java. 
To run the tests :
In eclipse Project explorer- 
1)First right click on the project "RhineSignupBDD" then click on run as Maven Test(First time)
2)After the project is build ,Right click on feature file "signUp.feature" and then click on "Run Configurations"
3) In "Run Configurations", select SignUp.Feature under cucumber feature in the left pane ,followed by "Run"
